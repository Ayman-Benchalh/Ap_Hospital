# 🏥 Ap_Hospital — Web-Based Clinic Appointment System

A web-based clinic/hospital appointment management system built with **PHP + MySQL** (HTML/CSS/JS on the frontend).
It supports multiple roles (Admin, Clinic, Doctor, Patient) and provides scheduling, appointment tracking, and basic clinic management.

---

## ✨ Key Features

### 👤 Roles
- **Admin**
  - Approve clinics (new registrations)
  - Manage specialties
  - View clinic / doctor / patient lists

- **Clinic**
  - Manage clinic profile & manager account
  - Add / edit / delete doctors
  - View appointment list & status
  - Publish announcements for patients

- **Doctor**
  - Manage doctor profile & password
  - Manage available schedule
  - Add symptoms / diagnosis / advice
  - Follow-up appointments
  - Manage treatment types

- **Patient**
  - Register & manage profile
  - Book / cancel appointments
  - View booking status
  - Search clinics & doctors (filters)
  - Rate & review (if enabled)

---

## 🧰 Tech Stack

- **Backend:** PHP
- **Database:** MySQL
- **Frontend:** HTML, CSS, JavaScript, jQuery, AJAX
- **Libraries:** Bootstrap, DataTables, SweetAlert2, Chart.js, FullCalendar (depending on pages)
